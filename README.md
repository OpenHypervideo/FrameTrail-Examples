# FrameTrail-Examples
Demos and Examples of FrameTrail Projects

### Usage 

- In your FrameTrail installation, replace your `_data` directory with one of the `[project]/_data` directories in this repository. 
- Log in with `demo@frametrail.org` : `demodemo`
